Introduction

It is very common that we need to manipulate the data for easier access when we need them.

Other than basic variable types such as string, number and boolean, array, object and collection are commonly used.

It is very often that the data obtained from database query is returned in the form of collection.

In the form of collection, it is troublesome for us to find a specifc data, hence it is better to be converted to Record.

convertCollectionToRecord()

Make sure that this is used on collection that has unique id, else some of the data will be lost.

Use case example:

            
    const dataFromDatabase = [
        {
            item_id: 'abc001',
            item_category: 'pen',
            item_name: 'Brand A',
            item_price: 2
        },
        {
            item_id: 'abc002',
            item_category: 'pen',
            item_name: 'Brand B',
            item_price: 3
        },
        {
            item_id: 'abc003',
            item_category: 'ruler',
            item_name: 'Brand X',
            item_price: 2
        }
    ];

    this.dataRecord = this.shareFacadeService.convertCollectionToRecord(dataFromDatabase, 'item_id')

    // dataRecord output
    {
        abc001: {
            item_category: "pen",
            item_id: "abc001",
            item_name: "Brand A",
            item_price: 2
        },
        abc002: {
            item_category: "pen",
            item_id: "abc002",
            item_name: "Brand B",
            item_price: 3
        }, 
        abc003: {
            item_category: "ruler",
            item_id: "abc003",
            item_name: "Brand X",
            item_price: 2
        }
    }

Then, we can easily access and modify any property of the items as long as we know the item_id

Example:

            
    // IN item-configuration.component.ts

    modifyPrice(itemID: string, newPrice: number) {
        this.dataRecord[itemID].item_price = newPrice
    }

    renameItem(itemID: string, newName: string) {
        this.dataRecord[itemID].item_name = newName
    }
