Introduction

Global styles can be defined in the styles.scss or styles.css in the src directory.

Then, all components in the src are able to use the styles declared.

flexContainer, flexContentSize, flexRemainingSpace

The "flexContainer" is used to wrap a division where "flexContentSize" and "flexRemainingSpace" will be used.

The "flexContentSize" will span the space according to the size of the content.

The "flexRemainingSpace" will fill up the remaining space in the flexContainer.

Example:

            
                

                    

                    

                

    <div class="flexContainer" style="background: black; height: 100px;">
        <div class="flexContentSize" style="height: 20px; background: rgb(53, 175, 134);"></div>
        <div class="flexRemainingSpace" style="background: rgb(97, 86, 250);"></div>
    </div>

app-spacer

This is used to fill up the empty space between two end of elements.

Example:

            
                
                    
                    My App
                    
                    
                
    <mat-toolbar>
        <button mat-icon-button>
            <mat-icon>menu</mat-icon>
        </button>
        <span>My App</span>
        <span class="app-spacer"></span>
        <button mat-icon-button>
            <mat-icon>favorite</mat-icon>
        </button>
    </mat-toolbar>

pre code

This is the style used to display the html snippets.

Note that you need to change the following

- Change '&' to '&'

- Change '<' to '<'

Example:

            
    <div class="flexContainer" style="background: black; height: 100px;">
        <div class="flexContentSize" style="height: 20px; background: rgb(53, 175, 134);"></div>
        <div class="flexRemainingSpace" style="background: rgb(97, 86, 250);"></div>
    </div>

    &lt;div class="flexContainer" style="background: black; height: 100px;">
        &lt;div class="flexContentSize" style="height: 20px; background: rgb(53, 175, 134);">&lt;/div>
        &lt;div class="flexRemainingSpace" style="background: rgb(97, 86, 250);">&lt;/div>
    &lt;/div>
