Introduction

Angular Material is the npm package of Material Design UI components for Angular applications.

Please refer to this site for the documentation about the available Angular Material Components.

Installation

npm i @angular/material

Code Snippet

            
    // IN resources.module.ts
    
    import { MatCardModule } from '@angular/material/card'; // only import the specific Angular Material module
    @NgModule({
        imports: [
            MatCardModule
        ]
    })
    export class ResourcesModule { }



    // IN angular-material-resources.component.html

    <div fxLayout="column" fxLayoutGap="10px">
        <mat-card>
            <h2>Introduction</h2>
            <p>Angular Material is the npm package of Material Design UI components for Angular applications.</p>
        </mat-card>
    </div>
