Introduction

Angular Flex-layout provides a handful of attributes that can be used in HTML tags for us to handle different layouts and sizing easily.

It also can be used for handling grid and responsive layouts, refer to this site for an overview of the features

Installation

npm i -s @angular/flex-layout @angular/cdk

Code Snippet

            
    

        

            
            
        

        

            
            
        

        

            
            
        

    

    <div style="background: burlywood; " fxLayout="row" fxLayoutAlign = "center center" fxLayoutGap="50px">
        <div style="background: cadetblue; padding:10px; height: 300px; width: 200px;" fxLayout="column" fxLayoutAlign = "start center" fxLayoutGap = "20px">
            <button mat-raised-button>Button 1</button>
            <button mat-raised-button>Button 2</button>
        </div>
        <div style="background: cadetblue; padding:10px; height: 300px; width: 200px;" fxLayout="column" fxLayoutAlign = "end end" fxLayoutGap = "20px">
            <button mat-raised-button fxFlexAlign="end end">Button 3</button>
            <button mat-raised-button>Button 4</button>
        </div>
        <div style="background: cornflowerblue; padding:10px; height: 300px;" fxLayout="row" fxLayoutAlign = "start end" fxLayoutGap = "30px">
            <button mat-raised-button>Button 5</button>
            <button mat-raised-button>Button 6</button>
        </div>
    </div>
