Introduction

It is very common that we need to manipulate array, collection and object in order to retrieve the data we want.

Lodash-es is a ES module of Lodash package which provides some great modular functions for us to manipulate arrays, objects, numbers, strings, etc.

This article explains why lodash-es is preferred over lodash

Refer to this link for Lodash Documentation, it can be used for both lodash and lodash-es.

Installation

npm i lodash-es

Code Snippet

            
    // IN item-list.component.ts

    import sortBy from 'lodash-es/sortBy';
